# Feelow
프론트엔드 파트 작업 공간입니다.
시작화면, 로그인, 추가정보 입력, 학생 메인페이지 챗봇, 학생 달력, 학생 상점, 마이페이지, 선생님 메인페이지를 구현하였습니다.
학생의 ai와의 대화를 통하여 학생들의 정신건강 상태 확인 및 교사의 학생관리, 대화를 통한 학생의 기분을 달력에 표시하여 학생들 상태 확인 가능한 플랫폼입니다.
